1 Introduction

1.1 Pyxel

Welcome to the Pyxel tutorials. Here we aim to give an introduction to creating games and other fun things with Pyxel.

Pyxel (/ˈpɪksəl/) is a retro game engine for Python. You can find more about Pyxel on the Github repository.

Pyxel on Github

The tutorials form a follow along project to create a whole game similar to a famous arcade classic!

1.2 Requirements

We assume only basic prior technical knowledge, as a minimum you should be able to

  • Sign into a computer running Windows or Linux and access the internet
  • Use basic system controls such as keyboard and mouse (or other pointing devices)
  • Install applications

It is recommended that you have some knowledge or experience of

  • Creating directories and files
  • Coding, in Python or similar

We will be using these applications

  • Command Line (cli) - Powershell on Windows and any default terminal emulator on Linux that comes pre installed
  • A code editor - VSCode which provides many extensions and inlcudes a teminal too

1.3 Topics covered in each section

  • Installation - Install of Python, Pyxel and VSCode for Windows and Linux
  • Hello World - the first Pyxel app and testing everything is working
  • Pyxel Edit Sprites - draw a main charachter sprite in the built in editor and make it move
  • Pyxel Edit Tilemap - draw a background for our sprite to move around in
  • Pyxel Edit Baddies - draw some baddie sprites to chase our main charachter around
  • Game mechanics - code the behaviour of our baddies and what happens when they get us
  • Screens - Splash, start, game over, credits…
  • Sound effects and Music
  • Publishing
  • Version Control
